Procedura per avviare il WebServer:

1) Importare il progetto in NetBeans con Maven.

2) Aggiungere ai servers un server APACHE TOMCAT v7

3) Lanciare il progetto "ticket" sul server appena aggiunto.

N.B: Quando viene lanciato il WebServer esso tenta di collegarsi al sistema centrale, che pertanto deve essere gia' attivato. L'indirizzo IP e la PORTA di questa connessione sono modificabili nella classe "Start.java" presente nel package "avvio".
